# ZMYTH

## What is this?

This is a crafting/merge game which I am developing in my spare time. With nothing but an infinite source of goo, can you zmyth together the foundations of technology and achieve true invention? Loosely inspired by [Minecraft](https://www.minecraft.net), [GregTech](https://gregtech.overminddl1.com/), and other [factory simulation games](https://en.wikipedia.org/wiki/Construction_and_management_simulation#Factory_simulation_games).

I have decided to call the game `ZMYTH` (pronounced "smith", if you like). This is awkward, as I already have a repo called [zmyth](https://github.com/breadsmyth/zmyth). Completely unrelated, please do ignore.
